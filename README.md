# RAG-ChartBot-project


An intelligent AI-powered web application that reads your PDF documents and answers questions with smart chart visualizations — powered by GPT, Google Gemini, and Groq API.

Features
📂 PDF Upload & Processing — Upload any PDF and the app extracts and indexes the content automatically
🔍 RAG (Retrieval-Augmented Generation) — Finds the most relevant sections of your document before answering
📊 Chart Generation — Visualizes data and answers in the form of charts and graphs
🧠 Multi-Model Support — Switch between GPT, Google Gemini, and Groq API
💬 Conversational Q&A — Ask questions naturally and get accurate, context-aware answers

Tech Stack
Frontend / UI - Streamlit
Language - Python 
AI Model - GPT OSS 120 B
API - Gorq API
Embeddings - OpenAI Embeddings / HuggingFace
